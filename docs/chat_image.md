
# 캐릭터 이미지 생성 방법 정리

## 1. 참고 자료
- 참고 영상: https://youtu.be/S4CamLlJXWY?si=rZHpf_lgcJk09bT3
- 이미지 레퍼런스: Pinterest에서 수집
- 캐릭터 특징: PRD 파일에 정의된 캐릭터 설명 활용

## 2. 전체 제작 흐름

1. 캐릭터 성격과 특징을 PRD에서 정의
2. Pinterest에서 캐릭터 스타일 및 레퍼런스 이미지 수집
3. 회사 고유 캐릭터임을 나타내기 위해 **편지 이미지 먼저 생성**
4. 생성된 편지 이미지를 **나노바나나(nano-banana)**에 레퍼런스로 활용
5. 편지를 들고 있는 캐릭터를 생성
6. 최종 캐릭터 이미지를 제작

## 3. 캐릭터 생성 전 편지 이미지 제작

회사 아이덴티티를 표현하기 위해 먼저 편지 이미지를 생성하였다.

### 사용 프롬프트

```
payletter 로고(2번째 사진)가 작게 적혀있는 편지를 만들어줘. 편지는 첫 번째 사진 앞 쪽에 있는 편지봉투 디자인 레퍼런스를 참고해줘. 배경없이 편지봉투 이미지만 만들어줘. payletter는 결제 회사이기 때문에 하트 대신 결제를 나타낼 수 있는 카드나 돈 모양으로 변경해줘.
```

### 목적

- 캐릭터가 들고 있을 **회사 상징 오브젝트 생성**
- 캐릭터가 **페이레터의 메시지를 전달하는 역할**을 표현

## 4. 캐릭터 이미지 생성

편지 이미지를 레퍼런스로 사용하여 캐릭터를 생성하였다.

### 사용 프롬프트

```
첫번째 레퍼런스 사진을 참고하고 하단의 캐릭터 설명을 참고해서 독수리 캐릭터를 만들어줘. 배경 없이 캐릭터만 만들어줘. 캐릭터는 앞서 보내준 편지를 들고 있도록 해줘. 아래의 캐릭터 성격에 맞게 포즈와 표정을 만들어줘. 이미지 테두리와 검정색 그림자 없애줘.
```

## 5. 핵심 생성 방식 요약

- **레퍼런스 이미지 기반 생성**
- **회사 아이덴티티 오브젝트(편지) 먼저 생성**
- **캐릭터 성격 기반 포즈 및 표정 생성**
- **배경 제거 및 캐릭터 중심 이미지 생성**
